书店介绍

一书屋位于山东省淄博市的山东理工大学，店铺成立以来发展迅速，业务不断发展壮大。书屋植基于推广阅读、激发创意、深耕文化、提升心灵的理念，致力于打造专注汉语国际教育及中华文化传播的专业书店品牌。成立以来，努力创造出既具本地特色、又具国际视野的文化景观。

一书屋始终认真贯彻执行党和国家的出版方针政策，遵守国家法律、法规，遵守《全国书刊发行业公约》，不经营盗版、假冒、伪书等非法出版物，自觉维护市场秩序。在经营方面从无违法、违规、违章的处罚记录。始终按照《争创首都规范化服务行业》和《中国书店服务质量基础规范管理标准》的要求实行科学、规范管理，落实《中国书店关于各类出版物政治安全审核制度》、《中国书店安全保卫工作规则》等企业管理制度，积极参与公益活动。
